Project Overview

This project focuses on analyzing chocolate sales data using Python in Google Colab. The aim is to understand sales performance, identify top-selling products, and generate useful business insights.

Project Objectives

To analyze chocolate sales data
To identify top-performing products
To calculate revenue and sales trends
To visualize data using charts
To generate meaningful business insights

Project Steps

1. Data Collection

The dataset named "Chocolate Sales.csv" was uploaded and used for analysis in Google Colab.

2. Data Cleaning

Duplicate values were removed and missing values were handled using appropriate methods like forward fill.

3. Data Exploration

The dataset was explored using functions like head(), info(), and describe() to understand its structure and content.

4. Feature Engineering

A new column called "Revenue" was created by multiplying Quantity and Price.

5. Data Analysis

Data was grouped and analyzed to find top-selling products and total revenue generated.

6. Data Visualization

Different types of charts such as bar graphs, histograms, and heatmaps were used to visualize the data.

7. Insights Generation

Key insights such as best-selling products, revenue trends, and region-wise sales were identified.

8. Output

The processed dataset was saved as "Processed_Chocolate_Sales.csv".

Conclusion

This project successfully analyzed chocolate sales data and provided useful insights that can help improve business decisions and sales strategies.
